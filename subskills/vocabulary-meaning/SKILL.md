# Vocabulary Meaning Sub-skill / 词义题子模块

Use for isolated English words, vocabulary screenshots, word lists, and explicit “词义题/单词题” requests.

- Use `type: QA` and `question_type: vocabulary_meaning`.
- Ask for the English learning stage when it is not already known.
- Put a contextual sentence with the target word emphasized on the front.
- Require dictionary meaning, part of speech, Chinese gloss, level, phonetic, audio, Oxford/Collins notes, three bilingual examples, vocabulary relations, and other meanings with examples.
- Never downgrade an isolated word to generic `short_answer` or `cloze`.
- Apply the shared two-stage preview and explicit-import gate.
