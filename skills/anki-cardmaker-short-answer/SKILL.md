# Anki Short-Answer Cards

- Generate `type: QA` with `question_type: short_answer`.
- Keep each question focused and each answer concise but sufficient.
- For Chinese non-language-learning topics, keep the question, answer, explanation, and mnemonic in Chinese whenever possible.
- Follow the shared implementation and two-stage import gate in `anki-cardmaker/SKILL.md`.
